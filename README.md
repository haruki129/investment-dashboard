# Dual-Eye Cockpit v1.2 — RACE MODE

車ダッシュボードメーター×レーストラック進捗を融合した投資インテリジェンス。

## v1.2の新機能（v1.1→v1.2）

- 🏁 **銘柄ごとにSTART→GOALのレーストラック進捗バー**
- 🏎️ **現在値を表す車アイコンが進捗に応じて移動**
- 🛫🏃⚡🏁 **4段階ステータス**（STANDBY / 走行中 / 加速中 / GOAL到達）
- 📏 **自動START/GOAL算出**（20日移動平均 / 52週高値ベース）
- 🚨 **GOAL到達アラート**自動発生
- ⚡ **加速中銘柄の優先通知**

## 見方（レーストラック）

```
🏁 START ━━━━━━🏎️━━━━━━━━━━ GOAL 🏆
$120.00              $142.35     $180.00
          [======37%          ]
🏃 走行中
```

- **START旗**：買い検討価格水準
- **🏎️ 車アイコン**：現在値の位置
- **GOAL旗**：目標利確価格
- **％**：進捗（0% → 100%で目標到達）

## ステータス状態

| 状態 | 進捗 | 意味 |
|------|------|------|
| 🛫 STANDBY | マイナス | START未到達・様子見 |
| 🏃 走行中 | 0-69% | 目標へ向かって上昇中 |
| ⚡ 加速中 | 70-99% | GOAL目前 |
| 🏁 GOAL到達 | 100%+ | 利確検討ゾーン |

## ファイル
- `index.html` — UI本体
- `style.css` — ダッシュボード＋レーストラックCSS
- `app.js` — SVGゲージ＋進捗描画ロジック
- `data/*.json` — START/GOAL付き銘柄データ
- `scripts/fetch_data.py` — 自動計算バックエンド
- `.github/workflows/update.yml` — 6時間ごと自動更新
