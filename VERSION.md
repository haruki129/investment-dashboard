## v1.2.0 — 2026-04-19（RACE MODE）

### 🏁 v1.2の新機能
- **銘柄ごとのSTART/GOAL可視化**：レーストラック風進捗バー
- **現在値カーアイコン**：🏎️ が START → GOAL の間を動く
- **ステータス判定**：
  - 🛫 **STANDBY**（START未到達）
  - 🏃 **走行中**（進捗 0〜69%）
  - ⚡ **加速中**（進捗 70〜99%）
  - 🏁 **GOAL到達**（進捗 100%超・点滅表示）
- **自動START/GOAL算出**（Pythonスクリプト側）
  - START = 20日移動平均（買い検討水準）
  - GOAL = 52週高値 × 95%（現実的目標）
  - Progress = (現在 - START) / (GOAL - START) × 100%
- **GOAL到達銘柄の自動アラート**（警告パネルに表示）
- **加速中銘柄の優先アラート**（進捗70%以上）

### 🎨 デザイン継承（v1.1そのまま）

- 2スピードメーター対決
- 4燃料計風ミニゲージ
